# rock_paper_scissors
Однопользовательская консольная игра "Камень, ножницы, бумага"

## Запуск игры

Игра не требует установки дополнительных библиотек (не имеет зависимостей). Все используемые в коде библиотеки встроены в базовый пакет Python3

Для запуска: 
* склонируйте данный репозиторий
* Перейдите в корневую деррикторию проекта
* выполните команду `python3 game.py`

## Краткое описание

1. В меню игры можно выбрать количество раундов. 
2. По завершению игры будет напечатан результат игры и счет, затем выполнен автоматический переход в главное меню.
3. Результат каждой игры записается в фаил `score_log.txt` в корне проекта. Если файл отсутствует - он будет автоматически создан. 
4. Результаты каждой новой игры будет записан в конец файла `score_log.txt` (прошлые результаты не удаляются).
